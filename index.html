Folksy

Folksy is a suite of learning games and a framework for creating such games. The project aims to create:

  • A common format for simple matching games.
  • Game engines on various platforms (web, desktop, mobile) for this format.
  • A set of free-content games based on media from Wikimedia Commons and other Wikimedia projects.
  • A simple way for users to create own, personalized games.
